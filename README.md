# 🐾 Pet Care Website

Welcome to the **Pet Care** website — a clean, responsive, and informational site made using **HTML** and **CSS**, hosted at:  
[abiraisingh.github.io/pet-care/](https://abiraisingh.github.io/pet-care/)

---

## 🌟 Overview

This is a simple static website to provide pet-care information — such as tips, guidance, and general advice — in a friendly and accessible way. It’s designed to be easy to navigate and visually appealing.

---

## 📚 Features

- 🌐 Responsive layout (works on desktop and mobile)  
- 📝 Informational sections on pet care (e.g., feeding, grooming, health)  
- 📊 Styled content using CSS for clear readability  
- 🧼 Clean design: no frameworks, pure HTML & CSS  
- 🔧 Easy to maintain and extend

---

## 🛠 Technology Stack

- **HTML5** — for structure and content  
- **CSS3** — for styling and layout  
- **GitHub Pages** — for hosting the live site

---

## 📂 Project Structure
```

pet-care/
│── index.html
│── styles.css
│── assets/ # optional: images, icons, etc.
└── README.md

```

- **index.html** — The main HTML file that contains all sections/web pages.  
- **styles.css** — The CSS file for layout, typography, colors, and responsiveness.  
- **assets/** — Optional folder for any images or icons.
